# dod_color_bench

Bench script for Data Oriented Design.
Main project: [dod_color](https://github.com/ESGI-SCHOOL-5AL/dod_color) 

## Run Bench

```sh
# We assume that rust and cargo are installed and that your are at the root of project
cargo bench
```
